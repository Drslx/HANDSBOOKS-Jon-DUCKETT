
1. Cada elemento "form" requer um atributo "action" e seu valor e o URL
2. Formularios podem ser enviados com um destes dois metodos: "get ou post"
3. Para formularios curto, como caixa de pesquisa, recuperar dados do servidor usamos o método "get"
4. Para formularios longos, contiver dados sigilosos, adicionar ou excluir informacoes de banco de dados usamos o método "post"
